* The Four Phases of AI-Assisted Coding

** Phase 1: Copy-Paste Era

#+begin_export latex
\begin{center}
\begin{tikzpicture}[scale=0.8, transform shape,
    node distance=1.4cm,
    box/.style={draw, rounded corners, minimum width=2.2cm, minimum height=0.8cm, align=center, fill=gray!10, font=\small},
    arr/.style={-{Stealth[length=2.5mm]}, thick}
  ]
  \node[box] (browser) {Browser\\ChatGPT};
  \node[box, right=of browser] (read) {Read\\Output};
  \node[box, right=of read] (copy) {Copy-Paste\\into Editor};
  \node[box, right=of copy] (fix) {Fix \& Pray};
  \draw[arr] (browser) -- (read);
  \draw[arr] (read) -- (copy);
  \draw[arr] (copy) -- (fix);
  \draw[arr, dashed, bend left=30] (fix) to node[above, font=\scriptsize] {repeat} (browser);
\end{tikzpicture}
\end{center}
#+end_export

- Open ChatGPT in browser, describe the problem
- Copy-paste code into editor, manually fix mismatches
- *No project awareness, no file access*

** Phase 4: Agent-to-Agent Protocols

#+begin_export latex
\begin{center}
\begin{tikzpicture}[scale=0.8, transform shape,
    node distance=1cm and 2cm,
    agent/.style={draw, rounded corners, minimum width=2cm, minimum height=0.8cm, align=center, fill=green!10, font=\small},
    tool/.style={draw, rounded corners, minimum width=1.8cm, minimum height=0.6cm, align=center, fill=yellow!15, font=\scriptsize},
    proto/.style={draw, thick, dashed, blue!60, rounded corners, inner sep=0.4cm},
    arr/.style={-{Stealth[length=2.5mm]}, thick}
  ]
  % Agents — wide gap so tools don't overlap
  \node[agent] (a1) {Agent A};
  \node[agent, right=6cm of a1] (a2) {Agent B};

  % A2A protocol between agents
  \draw[arr, blue!60, thick, <->] (a1) -- node[above, font=\scriptsize, blue!60] {A2A Protocol} (a2);

  % Tools below each agent — enough spread
  \node[tool, below left=0.7cm and 0.8cm of a1] (t1) {Tool 1};
  \node[tool, below right=0.7cm and 0.8cm of a1] (t2) {Tool 2};
  \node[tool, below left=0.7cm and 0.8cm of a2] (t3) {Tool 3};
  \node[tool, below right=0.7cm and 0.8cm of a2] (t4) {Tool 4};

  \draw[arr, gray] (a1) -- (t1);
  \draw[arr, gray] (a1) -- (t2);
  \draw[arr, gray] (a2) -- (t3);
  \draw[arr, gray] (a2) -- (t4);

  % Labels
  \node[below=0.2cm of t1, font=\scriptsize, gray] {MCP (agent $\leftrightarrow$ tool)};
  \node[above=0.4cm of $(a1)!0.5!(a2)$, font=\scriptsize, blue!60] {Agent $\leftrightarrow$ Agent};
\end{tikzpicture}
\end{center}
#+end_export

- MCP: how agents discover and use tools
- A2A: how agents communicate with each other
- Agents buying services from other agents
- *Next: make your services discoverable by agents, not just humans*
